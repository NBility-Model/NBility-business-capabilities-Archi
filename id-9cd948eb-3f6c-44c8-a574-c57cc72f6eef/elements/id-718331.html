Delen van netbeheerdata faciliteren ()
Het vermogen om het delen van netbeheerdata te faciliteren. In BAS/MFF model worden “uitvoeren markt(systeem)processen” en “data delen” onderscheiden. De wet kent vier soorten data delen: voor markt(systeem)processen, data delen op verzoek (toestemming), dataverzoeken van derden en open data. Deze capability vult in het data delen op verzoek (toestemming), dataverzoeken van derden en open data.

Name (en)Enable grid operator data distribution
Documentation (en)The ability to facilitate grid operator data sharing. In the BAS/MFF model, a distinction is made between “executing market (system) processes” and “sharing data”. The law recognizes four types of data sharing: for market (system) processes, data sharing on request (consent), data requests from third parties and open data. This capability concerns data sharing on request (consent), data requests from third parties and open data.
